Questão única:
Faça um projeto de um sistema neural de acordo com o seguinte roteiro.

1. Objetivo: descreva o que o sistema deve fazer, incluindo a área de atuação (por
   exemplo: sistema de controle, sistema de reconhecimento de padrões, sistema para
   visão, sistema biomédico, etc).
2. Justificativa: dê uma justificativa para a utilização de uma rede neural para solução do
   problema proposto (por exemplo: o sistema é não linear, apresenta parâmetros que
   devem ser regulados heuristicamente, não possui modelo matemático, possui um caráter
   intrinsecamente paralelo e distribuído, a quantidade de informação a ser tratada é muito
   grande, não existe conhecimento disponível, etc.)
3. Descrição do problema: apresente em detalhes o problema que deve ser tratado, de tal
   forma que seja possível identificar os componentes que irão constituir o sistema neural.
4. Construção e implementação do sistema neural: monte a rede neural proposta,
   identificando as entradas/saídas, o número de camadas escondidas, o tipo de rede, a
   função de ativação, etc. Indique o método de treinamento que será utilizado (Hebb,
   retropropagação do erro, clusterização, mínimos quadrados, etc.), justificando sua
   escolha. Se não houver, determine como será calculada analiticamente a matriz de
   ponderação da rede. Justifique suas escolhas.
   OBS: Todos os itens citados acima devem estar presentes na resposta da questão, ou o
   modelo não será considerado.
5. Análise dos resultados: Avalie os resultados obtidos, incluindo métricas pertinentes à
   área de redes neurais e à área do problema e as ferramentas de análise usadas para
   avaliar o desempenho do modelo. Discuta as diferenças de desempenho nas fases de
   treinamento, validação e testes. Use gráficos e tabelas para realizar a análise de
   desempenho.
   OBS: Todos os itens citados acima devem estar presentes na resposta da questão, ou o
   modelo não será considerado.
6. Conclusões: Comente as dificuldades encontradas, os resultados esperados (ou
   conseguidos) e as vantagens pretendidas com tal sistema.
7. Bibliografia.
